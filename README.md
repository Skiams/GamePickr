# GamePickr# GamePickr

GamePickr is a web app that recommends which game to play based on the user's backlog, available time, mood, and preferences.

## Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS

### Backend
- Node.js
- Express
- TypeScript
- Prisma

### Database
- PostgreSQL

## Features

- Add games to a personal backlog
- Store estimated game durations
- Pick a game based on available time
- Filter recommendations by mood and genres
- Explain why each game is recommended
